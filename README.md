### vue 单页面项目卸载依赖和插件的方式
方式1. 在package.json文件中删除相关的依赖和插件，然后重新运行 npm install
方式2. 在控制台运行 `npm uninstall 依赖名`，或者在 `vue ui` 打开图形界面的依赖部分点击删除按钮
